# 📋 React TodoApp

A modern and responsive **To-Do List App** built with **React JS** and **Tailwind CSS**.  
This project demonstrates key React concepts including state management, component architecture, props, and a clean UI/UX.
 
---

## 🌟 Live Demo
Check it: 
https://inspiring-salamander-edf796.netlify.app/

---

## 🚀 Features

- Create, update, and delete tasks (CRUD)
- Mark tasks as completed
- Modern dashboard-style UI using Tailwind CSS
- Component-based architecture
- Easy to enhance (filters, local storage, dark mode)

---

## 🧱 Tech Stack

| Technology | Purpose |
|------------|---------|
| React JS   | Frontend UI |
| Tailwind CSS | Styling & layout |
| Vite        | Development tooling |
| JavaScript  | Logic & state management |

## 📌 Getting Started (Local Setup)

### 1. Clone the repository
```bash
git clone https://github.com/Ahmed-Devx/TodoApp-React.js
````

### 2. Navigate to project folder

```bash
cd TodoApp-React.js
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run in development mode

```bash
npm run dev
```

📌 Open in browser:

```
http://localhost:5173
```
